# **安全作業基準**

**作成日:** 2024年 11月 21日  
**部署:** 横浜子供部屋研究所 ブロック開発Gr

| 項目 | 内容 |
| :---- | :---- |
| **分類記号番号** | L-001 |
| **作業工程** | **レゴデュプロによる「赤青交互タワー」の建設** |
| **特記事項** | （踏みつけ事故防止、誤飲防止の徹底） |
| **緊急時の措置** | 作業中断、直ちに保護者へ報告 |
| **使用器具** | レゴデュプロ（赤・青ブロック）、基礎板（緑） |
| **工具** | ブロック外し（セパレーター） |
| **保護具** | **ルームシューズ（常時）、保護メガネ（解体時）、綿手袋** |
| **人員** | 1人～2人 |

### **作業手順詳細**

| No. | 作 業 手 順 | 急 所（項目・頻度・条件等の要点・勘所・過去の災害例） | 理 由 |
| :---- | :---- | :---- | :---- |
| **【入室～準備】** |  |  |  |
| ① | 子供部屋入室SOPに従って入室（足元の安全確認） | 床に他の玩具が散乱していないか確認する | 転倒およびブロック紛失防止のため |
| ② | 使用するブロックを選別し、作業エリアに配置する ・基礎板：緑色（大）×1枚 ・赤ブロック：2×4ポッチ（長方形）×4個 ・青ブロック：2×4ポッチ（正方形）×4個 | **類似の「レゴクラシック（小）」が混入していないか確認** | デュプロ（大）とクラシック（小）は結合互換性が限定的であり、誤飲リスク管理のため |
| ③ | 基礎板を平坦な床または机の上にセットする | 絨毯の上など不安定な場所を避ける | 構造物の倒壊を防ぐため |
| **【組立～検査】** |  |  |  |
| ④ | 基礎板の中央に「赤ブロック（2×4）」を2個並べて接続する | **「カチッ」と音がするまで押し込む** | 結合不十分によるグラつきを防止するため |
| ⑤ | 1段目の上に「青ブロック（2×4）」を2個をまたぐように配置する | 赤ブロック同士の継ぎ目を青ブロックで跨ぐように接合する | 継ぎ目が揃う（芋目地）と強度が著しく低下するため |
| ⑥ | 赤と青を交互に積み上げる。全部で4段積み上げる。 | **15cm以上積み上げる際は、ブロックの歪みを目視確認** | 傾きがある状態で積み続けると倒壊の危険がある |
| ⑦ | 指定高さに到達後、タワー頂部を軽く指で押し、強度を確認する | 過度な力は加えないこと | 急激な荷重による破損が無い事を確認する |
| **【解体～退室】** |  |  |  |
| ⑧ | 解体作業を開始する（上段より順に取り外す） | **保護メガネを着用する** 無理に引っ張らず、ブロック外しを使用 | 勢いでブロックが弾け飛び、目に入る事故を防ぐため |
| ⑨ | 全てのブロックを回収し、個数確認を行う | ベッドの下、机の裏などを確認する | **踏みつけによる足裏への激痛災害（労災）を防ぐため** |
| ⑩ | 所定の収納ボックスへ色別に分別して保管する | 次回作業の効率化のため |  |
| ⑪ | 退室SOPに従って退室（照明消灯、ドア閉） |  |  |
